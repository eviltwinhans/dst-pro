Обзор

Проект #4 заключается в анализе данных о перелётах по России 
и о работе с этой базой данных для того, чтобы:
1. Применить базовые операторы SQL
2. Поработать с базой данных, соединением и объединением таблиц.
3. Научиться самостоятельно подбирать данные для аналитической задачи. 

Данные 

https://postgrespro.ru/docs/postgrespro/10/demodb-bookings

Описание файлов

sql_queries_project.txt - SQL-запросы для решения задач из модулей 4 и 5, а также итоговый запрос проекта
project_query - итоговый SQL-запрос проекта отдельно, сохранён из pgAdmin 4
data_winter_16-17.csv - итоговый датасет, полученный при помощи запроса project_query
